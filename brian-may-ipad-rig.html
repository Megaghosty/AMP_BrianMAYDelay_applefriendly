Prêt à démarrer

Dans Safari, touche Partager puis Ajouter à l'écran d'accueil pour obtenir un lancement plein écran type app.

Démarrage

Sur iPad, active d'abord le micro puis le moteur audio via ce bouton.

Conseils iPad

Utilise de préférence Safari, un casque ou une interface audio, puis ajoute l'app à l'écran d'accueil. Si le son se coupe après un changement d'app, relance le moteur audio avec le bouton dédié.
Preset storage en mémoire locale si disponible, sinon session en cours seulement.
